Google Software Product Sprint
'Automated Event Suggestion using Natural Language Processing Model'
Project Duration: 2 weeks
Brief
1.   Under the guidance of a Project Advisor and Technical Coach, both of whom are employed as Software Engineers at Google, I created a Natural Language Processing Model which classifies chat messages.
2.   The scope of this project was 2 weeks.
3.   The model is embedded within a greater project idea: an event bot for group chats. The bot, abstracted to its full extent, was a binary classifier, taking as input features of messages and returning ‘yes’ or ‘no’ whether the message alluded to participants attempting to scheduling an event (our definition of event is defined in a broad sense).
4.   Although our Event bot was made in this case for Discord, the model I proposed was abstracted enough that it could be implemented into any generic group chat interface (Snapchat, Facebook Messenger, iMessage, etc.)
5.   The proposal and creation of the model is solely mine. My group members contributed to rough project ideation and implementing this model within discord.

My full detailed paper outlining the model is here and the github to its implementation here.
